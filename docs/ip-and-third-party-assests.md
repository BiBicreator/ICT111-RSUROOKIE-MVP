## IP Summary

The RSU Campus Buddy project uses several third-party resources to support development, including Google Teachable Machine for AI image recognition, Bootstrap for responsive user interface design, Font Awesome for icons, Google Fonts for typography, and ChatGPT as a development assistant. These resources are used under their respective licenses or educational terms.

The project dataset, JSON database, HTML/CSS/JavaScript source code, and most campus building photographs are created by the project team for educational purposes. The team will ensure that all external resources are properly credited where required and that no copyrighted or licensed materials are used without permission. All AI-generated content is reviewed and edited by the team before inclusion in the final prototype.
