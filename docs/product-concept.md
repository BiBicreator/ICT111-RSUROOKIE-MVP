# Product Concept

## 1. Project Title

RSU Campus Buddy (Smart Campus Chatbot for Student Services with Image Recognition)

## 2. Target User

Primary User:
- Freshmen (1st-year students)
- International students

Secondary User:
- Current RSU students
- Campus visitors

## 3. Problem Statement

Freshmen and international students at Rangsit University often have difficulty identifying campus buildings and finding information about campus facilities, services, operating hours, student clubs, and university regulations. Existing information is scattered across different sources, making it inconvenient and time-consuming to access.

## 4. Proposed IT Product

RSU Campus Buddy is a web-based smart campus assistant that allows users to upload a photo of a campus building or search by building name. The system uses a Google Teachable Machine image recognition model to identify campus buildings and displays information stored in the project's own database, including available services, facilities, operating hours, student clubs, and campus regulations. Users can also interact with a chatbot to ask campus-related questions.

## 5. Core Value Proposition

RSU Campus Buddy provides students with a fast and centralized way to access campus information through image recognition, search, and chatbot features. It reduces the time spent searching for information and improves the campus experience for new and international students.

## 6. MVP Scope

The MVP will include:

- Homepage with search and image upload
- Building identification using a Google Teachable Machine image recognition model
- Search buildings by name
- Building information page
- Chatbot for frequently asked campus questions
- Admin page to add, edit, and delete building information
- Custom database containing campus building information
- Responsive web interface

## 7. Out of Scope for Current Semester

The following features will not be included in the MVP:

- Integration with official RSU or university databases
- Real-time GPS or indoor navigation
- Real-time building occupancy information
- Student login and authentication
- Personalized recommendations
- Multilingual support
- Automatic synchronization with university information systems
