## Reflection

### Which feature creates the strongest user value?

The **Upload Building Image & AI Recognition** feature creates the strongest user value because it allows students, especially freshmen and international students, to identify campus buildings quickly and access accurate information without searching multiple sources.

### Which feature creates the strongest business or operational value?

The **Admin Building Management** feature provides the strongest operational value because it enables administrators to maintain accurate and up-to-date campus information, ensuring reliable data for all users.

### Which required feature is still weak or unclear?

The **Building Directions** feature is currently limited because the MVP does not include real-time navigation or GPS integration. Instead, the system will provide static campus maps and basic directional information using the custom database.
